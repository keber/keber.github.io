# keber.github.io

Portfolio en React + Vite + TypeScript + Tailwind CSS.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

El resultado queda en `dist/`.

## Publicación en GitHub Pages

1. En GitHub, entra al repositorio y abre `Settings > Pages`.
2. En `Build and deployment`, selecciona `Source: GitHub Actions`.
3. Haz push a `main`.
4. El workflow `Deploy to GitHub Pages` construirá y publicará automáticamente.

Si este repositorio es `keber.github.io`, el sitio quedará publicado en:

`https://keber.github.io/`
